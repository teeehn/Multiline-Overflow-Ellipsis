# Multiline-Overflow-Ellipsis
jQuery Plugin. Adds overflow ellipsis to multi-line content.

## Usage:
```
  $([selector]).ellipsis()
```

## Note
Selector points to element(s) containing text.
  Text containing element should have css set to overflow: hidden.
